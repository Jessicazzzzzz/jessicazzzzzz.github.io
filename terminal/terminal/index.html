#tmux 简单命令使用

  • 新建窗口
  • 退出
  • 再回去
  • 分离
  • 查看有多少窗口
  • 杀死某个会话
  • 重命名
  • 切换
  • 杀死某个会话
  • 重命名

新建(在原terminal)

tmux new -s name

退出

exit

再回去

tmux attach -t name

分离

进程还在，后台执行

tmux detach

查看有多少窗口

tmux ls

杀死某个会话

tmux kill-session -t name

重命名

tmux rename-session -t name changedname

切换

tmux switch -t name

按住control+b ,再按d ，是退出，相当于detach
