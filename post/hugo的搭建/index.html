第一次搭建blog

Sep 5, 2021
1 min read
hugo

当你需要更新文章，通过git仓库放文章到博客hugo上的时候 1:第一步：在根目录下（即包括你所有文档的最外面一层文档）：hugo server -t 主题名字 –buidDrafts （这个在本地就建立了blog，可以在localhost:1313访问） 2:第二步：在根目录下，hugo –theme=“主题名字“ –baseUrl=“https://你自己的git仓库.github.io/” –buildDrafts (这样就生成一个public 文档) 以下都是在public 目录下操作 3:第三步： 进入public 目录，git init 4:第四步： git add . 5:第五步： git commit -m “自己起名字“ 6:第六步：git remote rm origin(我第一创建有这个，第二次提交就出错，必须remove) 7:第七步： git remote add origin https://github.com/你自己的github用户名/你的仓库名.github.io.git 8:第八步：git push -u origin master(我原本是main，但是我也不知道怎么改成了master，所以我有两个brunch，导致我还需要一步把master的pull到main上去) done!!
