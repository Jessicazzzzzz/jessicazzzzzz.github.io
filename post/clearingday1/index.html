cLearningDay1

Sep 5, 2022

注释符

  • //
  • /// 文档注释，用于注释函数和类
  • /* */

快捷键

  • ctrl +k+c 注释
  • ctrl + k+u 取消注释

变量

变量的类型

  • int ,char ,string ,decimal(m),double ,float(f) ,bool

  • char ,它是无法储存空字符的

  • decimal 是更加精确

  • 命名规范：变量开头小写，以后遇到一个单词大写;方法和类是每个单词开头都要大写

类型转换方式
  • 相同类型
    • 显性转换，大的存储类型->小的存储类型 ，（需要转换的类型）
    • 隐性转换，小的存储类型->大的存储类型,自动转换
  • 不同类型
    • 一般来说是string –>int ,面上过的去，过不去就需要用try-catch
    • Convert.ToDouble(string),Convert.ToInt32(string)

运算符

  • +只要一边遇到字符串，就是拼接，两边都是数字就相加
  • 只要做运算，必须保证两边的类型一致，包括复制运算

占位符

  • { 0 },跟c中很像，它的位置跟后面的变量是一样的，它相当于数组，需要按顺序走，小心越界

转义符

在读取文件的时候会很有效果

  • \b 回退一个 ⚠️ 如果放在开头，或者结尾，就不起作用
  • \n 回车 \r\n 是windows 系统中的回车
  • \t 一个tab 键
  • \ 可以后面跟" 也可以显示出来
  • @ 放在"c:\a\b “,这样就不需要写成"c:\\a\ \ b”
  • @还有一个功能是可以将原有的格式保留输出

异常捕获

  • 当一段代码出现问题的时候，就需要用try – catch
  • 注意catch 的范围，是到哪里，只要捕获了异常，就会不会执行有问题代码以后的任何代码内容

变量的作用域

  • {}外都不能使用

break

  • 跳出整个循环

switch- case

  • swithc(变量) 跟case “值” ，必须类型匹配

do -while/ while

continue

  • 跳出本次循环
