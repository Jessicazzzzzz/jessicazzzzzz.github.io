config.toml的一些设置

baseURL = "https://自己的域名"
languageCode = "en-us"
title = “博客的名字"
theme = "主题的名字"
[params]
author = "BALA"
description = "除了生死都是擦伤"
avatar ="girl.jpg"
  • 这样之后terminal 中输入会简单很多 1 .在根目录下按顺序输入以下的命令

hugo server -D
（建立本地站点）

hugo -D
（建立public文件）

2 . 在public 目录下

git add .
git commit -m "自定义名字"
git push

改变头像

  • 将照片放到theme目录里的static 里面
  • 将你根目录的config.toml 修改一下
[params]
author = "BALA"
description = "除了生死都是擦伤"
avatar ="girl.jpg"

出错情况

 unmarshal failed: toml: basic strings cannot have new lines
  • 我直接在vs code 中修改配置文件，就不行，必须用vi config.toml 修改，就不出错了。
