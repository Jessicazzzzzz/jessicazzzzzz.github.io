小标题

  2. “** **” 标题（一头一尾呼应）
  3. “1. 2. 3.” order list(每一行)
  4. “_ _“标题
  5. “* *” 斜体

链接

  1. “<>” link 的连接
  2. ”[] （）" =="[QUOTE]"+"()“需要紧跟在后面
