面向对象

先看下面的代码，是关于方法的运用



class Point {
    double x,y;
    public Point(double _x, double _y){
        x = _x;
        y = _y;
    }
    public  double getDistance(Point p){
        return  Math.sqrt((x-p.x)*(x-p.x)+(y-p.y)*(y-p.y));//注意优先级，需要大括号
    }

}
public class TestPoint{
    public static void main(String[] args) {
        Point p = new Point(3.0,4.0);
        Point origin = new Point(0,0);
        //理解这句
        System.out.println(p.getDistance(origin));

    }

}

每次调用一个方法，栈帧都会开辟一个空间，需要注意它什么时候会销毁，一般来说，方法调用结束，就会销毁

Point p = new Point(3.0,4.0);

  • 创建了一个新的对象，栈帧开了一个空间给它，默认值x=0,y=0，方法
  • 构造器重新开辟一个空间， 给x,y赋值为3.0，4.0
  • 当这句结束了，那对象调用就结束了，销毁x,y

Point origin = new Point(0,0);

  • 跟上面一样，这次又给x,y重新赋值为0，0
  • 这就结束时，销毁x,y

但是main方法没有结束，所以p,origin 的空间还在的

System.out.println(p.getDistance(origin));

这个比较像c的函数调用

  • 先调用对象p的getdistance()方法，在把origin 的值放进去
